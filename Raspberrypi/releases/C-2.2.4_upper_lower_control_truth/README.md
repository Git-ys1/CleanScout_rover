# C-2.2.4 上位-下位控制真相文档入口

## 本轮定位
- 关联任务：`C-2.2.4`
- 性质：真相文档冻结，不是功能开发
- 目标：把“树莓派上位控制 `UNO + AFMotor`”的联调真相和长期正式控制口径收敛成文档

## 现阶段脚本作用
- 当前树莓派侧脚本只负责验证：
  - `树莓派 -> UNO USB 串口`
  - `CSR_UNO_READY`
  - `ACK / ERR`
  - 单次动作链是否能被触发

## 当前协议定位
- 当前测试协议只是过渡联调协议。
- 当前 `1~9` 字符命令只是联调用动作码，不是长期正式接口。
- 当前 `4000ms` 类长超时只服务人工观察，不代表正式导航控制参数。
