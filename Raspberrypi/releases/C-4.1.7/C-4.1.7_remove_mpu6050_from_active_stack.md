## 3. 最终无 IMU 数据链

树莓派硬件顺序：

```text
清理旧会话
  -> roscore
  -> robot_state_publisher
  -> STM32 / RF1 底盘桥
  -> 等待 /rf1/vel
  -> RPLIDAR
  -> 等待 /scan
  -> cmd_vel_safety_gate
  -> 风机与顶盖
  -> edge-relay
```

PC 建图：

```text
/rf1/vel -> rf1_vel_to_odom.py -> /odom + odom->base_link
/scan + TF + /odom -> gmapping -> /map
```

PC 导航：

```text
/rf1/vel -> rf1_vel_to_odom.py -> /odom + odom->base_link
/map + /scan + TF + /odom -> AMCL -> move_base/TEB -> /cmd_vel_nav
/cmd_vel_nav -> cmd_vel_safety_gate -> /cmd_vel -> STM32
```

系统不发布假的 `/imu/data`，没有 IMU 话题是正常状态。
