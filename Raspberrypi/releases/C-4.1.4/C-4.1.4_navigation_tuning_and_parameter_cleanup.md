## 阶段结论

`C-4.1.4` 延续 TEB 默认导航基线，并封存本轮实车效果更好的 AMCL、
footprint、全局代价地图和 TEB 多拓扑参数。

本轮同时完成导航参数可读性治理。默认主链、传统回退链、AMCL、costmap、
RF1 几何入口和速度安全门均补充了中文注释，说明单位、调参方向、覆盖顺序
和跨文件耦合关系。

## 实车参数变化

### TEB 多拓扑

- `enable_homotopy_class_planning: false -> true`
- `max_number_classes: 2 -> 3`
- 新增 `max_number_plans_in_current_class: 2`
- `selection_prefer_initial_plan: 0.95 -> 0.75`
- `selection_obst_cost_scale: 1.0 -> 4.0`
- `selection_alternative_time_cost: false -> true`
- 新增 `selection_viapoint_cost_scale: 1.0`
- 新增 `selection_dropping_probability: 0.05`
- 路网采样数量 `12`
- 路网采样宽度 `1.5 m`

TEB `0.9.1` 的启动读取和 dynamic_reconfigure 使用不同的历史键名，
因此同时封存：

```yaml
roadmap_graph_samples: 12
roadmap_graph_no_samples: 12
```

### AMCL

- 里程计模型改为 `omni-corrected`
- 粒子范围由 `1000~6000` 收敛为 `500~2500`
- 激光抽样束数由 `90` 调整为 `60`
- 激光混合权重改为 `z_hit=0.85`、`z_rand=0.15`
- 里程计噪声改为 `0.12/0.08/0.12/0.08/0.18`
- 更新阈值改为 `0.20 rad`、`0.15 m`
- TF 容差改为 `0.5 s`
- 关闭随机粒子注入恢复

### 机器人轮廓与代价地图

- footprint 前缘由 `+0.218 m` 收回到 `+0.135 m`
- 当前轮廓为以 `base_link` 为中心的 `0.270 x 0.217 m` 矩形
- 全局 costmap 更新/发布频率改为 `1.0/0.5 Hz`
- 全局膨胀半径改为 `0.35 m`
- 全局膨胀衰减系数改为 `5.5`
- 局部 costmap 保持 `4.0/2.0 Hz`、`0.05 m` 膨胀半径

## 已知事项

1. 本轮不向树莓派同步带中文注释的文件。
2. 中文只存在于 YAML/XML 注释，不进入参数值；PC Noetic 端仍需通过
   UTF-8、XML、YAML 和 roslaunch 检查后发布。
3. footprint 与 C-4.0.1 早期几何记录存在阶段差异，C-4.1.4 以本轮实际
   导航效果采用的对称 footprint 为当前导航基线；后续若重做 URDF/碰撞
   模型，应重新统一物理外形、costmap 与 TEB 距离。
4. 多拓扑规划提高绕障能力，也增加 CPU 负载；长路线仍应继续观察控制周期
   和候选轨迹数量。
