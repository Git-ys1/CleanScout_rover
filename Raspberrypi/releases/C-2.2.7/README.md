# C-2.2.7 树莓派联调发布入口

## 本轮定位

`C-2.2.7` 是树莓派与 `UNO` 的协议纠偏轮。

本轮不再把树莓派当前脚本的 `CMD,vx,vy,wz` 当成联调真相，而是明确对齐到 `UNO` 当前已经实际实现的 `W,w1,w2,w3,w4`。

## 当前发布物

- `C-2.2.7B_pi_w_bridge.md`

## 当前真相

- `UNO` 当前真相协议：`W,w1,w2,w3,w4`
- `Pi` 当前必须对齐到 `W` 协议桥
- `CMD,vx,vy,wz` 只保留为旧桥协议历史说明

## 当前通过标准

- 收到 `CSR_UNO_READY`
- 发送 `W,...`
- 收到 `ACK:W`
- 能解析 `ENC,...`
- 能解析 `PID,...`
