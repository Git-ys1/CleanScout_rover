# CleanScout Maps

当前导航地图统一放在 ROS 包内，便于通过 `$(find clbrobot)/maps/...` 定位。

| 地图 | 分辨率 | 原点 | 当前用途 |
| --- | --- | --- | --- |
| [`407-5.22-2120.yaml`](407-5.22-2120.yaml) | `0.05 m/pixel` | `[-10.0, -21.2, 0]` | PC 导航默认地图 |
| [`406.yaml`](406.yaml) | `0.05 m/pixel` | `[-10.0, -10.0, 0]` | 早期 406 阶段地图 |
| [`407map.yaml`](407map.yaml) | `0.05 m/pixel` | `[-100.0, -100.0, 0]` | 大范围历史地图 |

## 注意事项

1. 每张地图由 `.yaml` 和同名 `.pgm` 组成，移动或发布时必须成对处理。
2. `407-5.22-2120.yaml` 与 `406.yaml` 当前记录了机器绝对路径；跨 PC/Pi 使用前应检查 `image`。
3. 新地图由 `catkin_ws/save_map.sh [name]` 保存，并在验证后更新本表。
4. 替换默认地图后要复查 AMCL 初始位姿、地图原点和障碍边界。
5. 大地图文件入库前应确认确有复现价值，避免把临时建图结果长期堆积。
