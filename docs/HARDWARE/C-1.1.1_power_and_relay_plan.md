# C-1.1.1 Power And Relay Plan

## 1. 文档目的

本文件用于冻结当前 `C-1.1.1` 版本下的供电关系和继电器风机方案，避免后续继续依赖口头解释。

## 2. 主供电关系

当前冻结口径如下：

- 主电源：`3S` 电池
- 主开关：船型开关
- 开关位置：位于主电源正极路径

从开关后的主电源正极，当前分为两条主要支路：

- 一路供给 AFMotor 电机电源侧，用于底盘四路电机
- 一路供给继电器触点侧，用于两路风机上电

当前文档口径中：

- 风机负极直接回系统 `GND`
- 继电器逻辑侧 `DC+ / DC-` 由 `5V / GND` 供电

## 3. 继电器模块冻结

模块冻结如下：

- 模块：`HW-803` 单路光耦隔离继电器
- 触发方式：高电平触发
- 控制口：`A2`
- 触点接法：`NO / COM` 用于切换两路风机正极

当前风机控制目标：

- 只做风机开关
- 不做风机调速

触发逻辑冻结为：

- `A2 = HIGH -> 继电器吸合 -> COM 与 NO 导通 -> 风机上电`

## 4. 风机方案冻结

两路风机当前按以下口径冻结：

- 两路风机正极并到继电器触点切换侧
- 两路风机负极直接回系统 `GND`
- 继电器作为总开关切换两台风机

说明：

- 当前方案只负责“开 / 关”。
- 当前方案不包含 PWM 调速、软启动或独立双路控制。

## 5. 风险与待确认项

以下事项在 `C-1.1.1` 仍属于待确认项，必须保留：

1. Motor Shield 电源跳线当前是否安装
2. UNO 逻辑电源当前是否与电机 / 风机共源
3. 两路风机并联总电流是否已测
4. 是否做过上电瞬态 / 复位测试

这些项目未确认前，不能默认当前供电方案已经完全稳定。

## 6. 当前风险备注

当前方案的工程风险主要有：

- 风机并联启动电流可能偏大
- 电机与风机共用主电源支路可能带来压降和瞬态干扰
- 若 UNO 逻辑电源与大功率负载共源，可能出现复位或通信不稳定
- 继电器方案只能做通断，无法提供调速与细粒度控制

## 7. 本轮冻结边界

`C-1.1.1` 只冻结当前供电与继电器方案，不代表：

- 已实现风机控制代码
- 已完成负载实测
- 已完成 EMI、压降或瞬态验证
- 已完成整车长时间稳定性验证
