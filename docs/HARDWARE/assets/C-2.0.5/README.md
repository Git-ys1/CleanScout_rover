# C-2.0.5 照片与草图提交说明

本目录用于存放 `C-2.0.5` 的硬件照片与草图留痕。  
本轮不要求接线图；这里只要求照片或草图证据。

## 最少提交内容

### 1. 底盘现状

- 当前整车俯视图
- 当前电机 / 轮组安装现状

### 2. 电机现状

- 当前驱动电机近照
- 轴径、孔位、外形尺寸可见证据

### 3. 预留安装位

- 雷达计划安装位
- IMU 计划安装位
- 编码器计划安装位
- 树莓派计划安装位

### 4. 计划接线位置

- 树莓派与 Arduino 的串口连接位置
- 编码器信号线进入 Arduino 的位置
- 雷达 USB 连接方向
- IMU I2C 连接位置

## 文件命名建议

- `chassis_overview_01.jpg`
- `motor_closeup_left.jpg`
- `motor_closeup_right.jpg`
- `rplidar_mount_plan.jpg`
- `imu_mount_plan.jpg`
- `encoder_mount_plan_left.jpg`
- `encoder_mount_plan_right.jpg`
- `serial_link_plan.jpg`

## 说明

- 可以是照片，也可以是手绘草图
- 重点是让后续实施人员能看见“现状、预留位、计划接线位置”
- 本轮明确不要求生成规范接线图
