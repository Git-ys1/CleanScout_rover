## 4. 当前测试流程

1. 串口上电后确认 `CSR_RF1_READY`
2. 单通道驱动测试：
   - `M,1,300` / `M,1,-300`
   - `M,2,300` / `M,2,-300`
   - `M,3,300` / `M,3,-300`
   - `M,4,300` / `M,4,-300`
3. 每步测试后执行：
   - `STOP`
4. 编码器查询：
   - `E,1` / `E,2` / `E,3` / `E,4`
5. 将结果回填到本表，不再靠口头记忆
