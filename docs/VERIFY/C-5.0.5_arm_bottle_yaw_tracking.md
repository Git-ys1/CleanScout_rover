# C-5.0.5：OrangePi bottle yaw-only 追踪阶段

## 目标

本阶段只做一件事：YOLO11 稳定识别 `bottle` 后，只控制 000 号底座 yaw，使目标水平回到画面中心。

暂不做：

- 不控制 003 pitch。
- 不做上下追踪。
- 不做抓取、IK 或 ROS 融合。

## 当前实现

- `target_selector.track_class = bottle`。
- `visual_servo.control_axes = [yaw]`。
- `driver.stop_servo_indices = [0]`。
- `yolo_arm_track.py` 新增 `--control_axes yaw` 参数。
- 视觉 PID 仍沿用 `visual_servo.py` 的 P/I/D 结构：
  - `kp_yaw = 0.0008`
  - `ki_yaw = 0.0`
  - `kd_yaw = 0.0`
  - `dead_zone_px = 30`
  - `max_yaw_delta = 0.015`
  - `control_rate_hz = 10`

## 通过标准

- 画面只把 `bottle` 作为 target。
- yaw-only 命令加 `--prepare_pose false` 后，启动、追踪、退出都只输出 000 号舵机命令。
- 左右移动 bottle 时，000 yaw 朝能减小 `err_x` 的方向动作。
- 目标进入水平死区后，000 yaw 不再明显来回抖动。

## 后续

- 若 yaw 方向反，先改 `driver.yaw_pwm_sign` 或 `visual_servo.invert_yaw`，不要同时改两个。
- yaw 稳定后再开启 `control_axes: [yaw, pitch]`，进入 003 pitch 上下追踪阶段。
