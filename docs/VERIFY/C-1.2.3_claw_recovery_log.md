## 6. 当前最重要的现场观察点

1. `claw_selftest.py` 是否能让夹爪肉眼出现 `OPEN -> CLOSE -> OPEN`
2. 当前 `60 / 150` 是否真的是有效开合值，还是语义反了
3. `MODE_FORCE_LOCAL_GRAB` 下是否即使没有识别和通信，也能看到夹爪闭合、抬升、放置、松开、回位
4. 若夹爪仍完全不动，应优先怀疑：
   - `B10 + Timer2 Channel3` 板级通道是否真实可用
   - 舵机供电/接线
   - 板级复用或卖家固件差异
