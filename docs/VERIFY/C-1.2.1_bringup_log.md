## 5. 当前已知问题

1. 还没有真实证明 `F411` 板上 `USART1/USART2` 已经成功收发，只是编译产物已经具备平台层。
2. `PB2` 状态灯码需要你上板后确认是否与文档一致。
3. `OpenMV` 当前抓取本体仍保留 vendor 阻塞式动作序列，本轮只解决等待态可观测和重发，不解决动作鲁棒性。
4. `UNO` 默认 boot 仍是 `MAN`，Gate4 需要通过 USB 串口先把车切入 `AUTO`，或先用 `MAN + 持续动作` 做过渡验证。

## 6. 下一步上板顺序

1. 先烧 `F411`，只看 `PB2` 灯码。
2. 短接 `PA2/PA3` 和 `PA9/PA10` 做单板串口回环。
3. 接 `UNO`，确认 `PING/PONG`、`STOP_REQ/STOP_ACK`、`RESUME_REQ/RESUME_ACK`。
4. 接 `OpenMV`，确认 `COLOR_FOUND -> PICK_WINDOW -> ARM_BUSY -> PICK_DONE/TIMEOUT`。
5. 最后做三板闭环和 10 轮重复性。
