# C-3.0.2D：X1 PI 重调轮

## Summary
- 本轮从 `C-3.0.2C` 的止损结论直接继续推进，不再把 `X2` 当当前主线。
- 当前唯一目标是：
  - 保持 `W,w1,w2,w3,w4` 协议不变
  - 保持当前新版底盘常量不变
  - 主线回到 `X1`
  - 按 `X1` 的测速颗粒度和当前轮间差异，重新调 `PI`
- 本轮通过标准不是“继续证明 `X2`”，而是：
  - 恢复并冻结 `X1` 下的三轮流畅同步转动

## Current Starting Facts
- 用户已明确确认：更早的 `X1` 阶段，三轮是可以流畅同步转动的。
- 当前把 `X2` 的 `PI / 补偿` 口径直接带回 `X1` 后，不能代表 `X1` 的真实能力。
- 因此 `C-3.0.2D` 应视为：
  - `X1` 的重新整定轮
  - 不是 `X2` 的继续试错轮

## PI Retune Direction
### 1. 本轮只围绕 X1 调
- `X1` 的测速颗粒度更粗
- 因此本轮整定优先考虑：
  - `Kp`
  - `Ki`
  - `I limit`
  - 分轮输出上限
  - 起步补偿
- 不再围绕 `X2` 的双边沿计数特性做补偿

### 2. 本轮目标
- 在 `W,300,300,300,300` 下：
  - 三轮可持续、流畅、接近同步转动
- 在必要时再用：
  - `W,400,400,400,400`
  做第二档验证

## Pass Criteria
- `X1` 口径保持稳定，不再被改回 `X2`
- `W,300,300,300,300` 下三轮恢复流畅同步转动
- `ENC / PID` 与肉眼观测一致
- 当前通过标准不要求 `X2` 通过
