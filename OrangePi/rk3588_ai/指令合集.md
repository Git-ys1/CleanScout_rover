# RK3588 AI 指令合集（精简版）

只保留当前机械臂追踪常用命令。C-5.0.6 阶段继续只追踪 `bottle`，默认只控制 001 抬升轴 `lift`；C-5.0.5 已验证的 000 底座 yaw 命令保留作回归。

香橙派固定 IP：`192.168.8.148`。

```bash
ssh orangepi@192.168.8.148
```

历史记录姿态：`0=1500,1=1907,2=1900,3=900,4=1500,5=1500`。如果实物方位不对，不要直接恢复这组值，先用交互模式重新调。

## 1. 进入环境

```bash
cd ~/rk3588_ai/arm_tracking_demo
source ~/rk3588_ai/rknn_lite_env/bin/activate
```

如从 SSH 启动但要把 OpenCV 窗口显示到远程桌面：

```bash
export DISPLAY=:0
export QT_QPA_PLATFORM=xcb
```

## 2. 终端交互调姿

```bash
cd ~/rk3588_ai/arm_tracking_demo
~/rk3588_ai/rknn_lite_env/bin/python3 tools/arm_servo_tune.py
```

进入 `servo>` 后常用：

```text
read 0-5
nudge 2=-50 1000
nudge 2=50 1000
nudge 3=-50 1000
nudge 3=50 1000
set 2=1900,3=900 1500
save 0-5
quit
```


只保存当前实物姿态：

如果你已经手动调到了一个好姿态，只想把它保存为追踪启动姿态，用这个。

```text
read 0-5
save 0-5
```

移动到历史记录姿态并保存：

注意：这条会先让 0~5 号舵机实际运动到下面这组固定 PWM，然后再把这组位置保存成启动姿态。它不是“保存当前实物姿态”。

```bash
cd ~/rk3588_ai/arm_tracking_demo
~/rk3588_ai/rknn_lite_env/bin/python3 tools/arm_servo_tune.py --set 0=1500,1=1907,2=1900,3=900,4=1500,5=1500 --duration_ms 1500 --settle_s 0.8 --save-start-pose --ids 0-5
```

## 8. 最小排障

```bash
# pyserial 缺失
~/rk3588_ai/rknn_lite_env/bin/python3 -m pip install pyserial

# 只测 yaw / pitch
cd ~/rk3588_ai/arm_tracking_demo
~/rk3588_ai/rknn_lite_env/bin/python3 tools/test_one_joint_yaw.py --serial_port /dev/ttyUSB0 --enable_arm --dry_run false --yaw 0.03
~/rk3588_ai/rknn_lite_env/bin/python3 tools/test_one_joint_pitch.py --serial_port /dev/ttyUSB0 --enable_arm --dry_run false --pitch 0.03
```

现场原则：先 `read 0-5`，再小步 `nudge`，确认能动后再跑追踪。不要一上来大幅改 `001/002/003`。
