# RK3588 AI 现场指令

只保留当前常用命令。香橙派固定地址：`192.168.8.148`。

## 1. 登录与设备

```bash
ssh orangepi@192.168.8.148
ls -l /dev/video* /dev/ttyUSB0
```

## 2. 交互调节总线舵机

```bash
cd ~/rk3588_ai/arm_tracking_demo && ~/rk3588_ai/rknn_lite_env/bin/python3 tools/arm_servo_tune.py
```

```text
servo> read 0-5
servo> set 2=1900,3=900 1500
servo> nudge 2=-50 1000
servo> nudge 3=50 1000
servo> save 0-5
servo> quit
```

`read` 不运动；`set` 是绝对 PWM；`nudge` 是相对增量；末尾数字为动作时间 ms；
`save` 只保存视觉追踪启动姿态。先读位置，每次只动一个关节并从 `20..50` 小步开始。

## 3. 标准姿态

- 视觉追踪：`1500,1907,1900,900,1500,1500`
- 固定视角抓取：`1380,1909,1900,620,1500,1500`

移动到固定视角抓取姿态：

```bash
cd ~/rk3588_ai/arm_tracking_demo && ~/rk3588_ai/rknn_lite_env/bin/python3 tools/arm_servo_tune.py \
  --set 0=1380,1=1909,2=1900,3=620,4=1500,5=1500 --duration_ms 5000
```

## 4. Bottle 视觉追踪

```bash
cd ~/rk3588_ai/arm_tracking_demo && ~/rk3588_ai/rknn_lite_env/bin/python3 yolo_arm_track.py \
  --model_path ~/rk3588_ai/models/official_yolo11.rknn \
  --target rk3588 --camera 0 --track_class bottle --control_axes yaw,lift,pitch \
  --serial_port /dev/ttyUSB0 \
  --dry_run false --enable_arm --print_cmd
```

只识别不运动：删除 `--control_axes/--serial_port/--enable_arm/--print_cmd`，改为 `--dry_run true --print_boxes`。

## 5. 固定视角标定

C-5.2.4 的 11 点矩阵已合格；相机、支架或底座未移动时不要重标。硬件改变后用新文件名采集：

```bash
cd ~/rk3588_ai/arm_grasp_pipeline && source ~/rk3588_ai/scripts/use_realsense_rsusb.sh
~/rk3588_ai/rknn_lite_env/bin/python3 tools/collect_base_camera_points.py \
  --prepare_reference_pose --serial_port /dev/ttyUSB0 \
  --output ~/rk3588_ai/calibration/base_camera_points_round2.csv
```

求矩阵并写配置：

```bash
~/rk3588_ai/rknn_lite_env/bin/python3 tools/calibrate_base_camera_3d.py \
  ~/rk3588_ai/calibration/base_camera_points_round2.csv \
  --output_json ~/rk3588_ai/calibration/base_camera_report_round2.json --write_config
```

合格条件：`quality_pass=true`、`rmse_m<=0.010`、`max_error_m<=0.015`。
