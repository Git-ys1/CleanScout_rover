# RK3588 AI 现场指令

只保留当前常用命令。香橙派固定地址：`192.168.8.148`。

## 1. 登录与设备

```bash
ssh orangepi@192.168.8.148
ls -l /dev/video* /dev/ttyUSB0
```

## 2. 交互调节总线舵机

```bash
cd ~/rk3588_ai/arm_tracking_demo && ~/rk3588_ai/rknn_lite_env/bin/python3 tools/arm_servo_tune.py
```

```text
servo> read 0-5
servo> set 2=1900,3=900 1500
servo> nudge 2=-50 1000
servo> nudge 3=50 1000
servo> save 0-5
servo> quit
```

`read` 不运动；`set` 是绝对 PWM；`nudge` 是相对增量；末尾数字为动作时间 ms；
`save` 只保存视觉追踪启动姿态。先读位置，每次只动一个关节并从 `20..50` 小步开始。

## 3. 标准姿态

- 视觉追踪：`1500,1907,1900,900,1500,1500`
```bash
  set 0=1500,1=1907,2=1900,3=900,4=1500,5=1500 1500
```
- 固定视角抓取：`1500,1909,1900,620,1500,1500`
```bash
  set 0=1500,1=1909,2=1900,3=620,4=1500,5=1500 1500
```

移动到固定视角抓取姿态：

```bash
cd ~/rk3588_ai/arm_tracking_demo && ~/rk3588_ai/rknn_lite_env/bin/python3 tools/arm_servo_tune.py \
  --set 0=1500,1=1909,2=1900,3=620,4=1500,5=1500 --duration_ms 5000
```

000 使用厂家/量角器比例 `8.148148 PWM/deg`；安装后的 001/002/003 使用实测
`7.090824/7.935827/6.478096 PWM/deg`，不能统一套 8.148。000 增大向左，001
增大向后，002 增大收肘，003 增大抬头，005 增大闭爪。005 使用 `1000` 全开、
`2000` 闭合；004 始终保持 `1500`，当前 STM32 命令上限为 `2490`。

## 8. 真实抓取分阶段

先把瓶子立在正前方，然后每次只改 `STAGE`。顺序固定为：
`open -> pre_grasp -> approach -> close -> lift`。

```bash
cd ~/rk3588_ai/arm_grasp_pipeline && source ~/rk3588_ai/scripts/use_realsense_rsusb.sh
STAGE=open
~/rk3588_ai/rknn_lite_env/bin/python3 tools/d435_yolo_grasp.py \
  --target_class bottle --execute_on_lock true \
  --dry_run false --enable_arm --joint_pwm_calibrated \
  --serial_port /dev/ttyUSB0 --max_stage "$STAGE" \
  --max_frames 600 --no_show
```

- `open`：回参考姿态并回读六轴，只把 005 张到 `1000`。
- `pre_grasp`：张爪后到瓶子前方 `70 mm`，不碰瓶子。
- `approach`：保持张爪，以 `10 mm` 路点水平到瓶心。
- `close`：到瓶心后把 005 闭合到 `2000`，不抬升。
- `lift`：完成闭爪并向上抬 `80 mm`。

正式配置固定 L3/TCP 水平：抓取 `pitch=0 deg`，LIFT 使用 `-11 deg`。按当前实测
连杆和舵机边界，瓶心距底座应至少约 `0.340 m`；若坐标验证输出 `plan_ok=false`，
先沿当前对正方向把瓶子向外移动，禁止用下俯 L3 或手调单轴 PWM 硬凑。
`--max_stage` 只预检到指定阶段，不会再被未来阶段无解错误阻断。

每次启动都会先回到 `[1500,1909,1900,620,1500,1500]` 并用 `PRAD` 核验；
任一轴偏差超过 `40 PWM` 会自动拒绝继续。详细说明见
`~/rk3588_ai/arm_grasp_pipeline/README.md`。
